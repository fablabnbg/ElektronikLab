GPIO-Pin-Charakteristik Eingänge

  • Bei CMOS (benutzen alle Prozessoren heutzutage) gilt für Eingangsspannungen:
  • U > 0,7*VCC → Eingang = 1
  • U < 0,3*VCC → Eingang = 0
  • Eingangsstrom < 1 µA
  • ACHTUNG: Spannungen kleiner -0,3V bzw. größer VCC+0,3V können das IC zerstören!
